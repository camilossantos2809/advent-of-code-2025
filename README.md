# Advent of Code 2025 - Go

Solutions for [Advent of Code](https://adventofcode.com/) challenges, implemented in Go.

Having fun solving challenges while learning Go and picking up some algorithm skills along the way.

## Structure

```
.
├── {year}/
│   ├── day{dd}/
│   │   ├── main.go
│   │   └── input.txt
│   └── ...
└── README.md
```

## Running

```bash
go run cmd/main.go {year} day{dd}
```
